¿Cómo escribir la Descripción?

El sistema detecta automáticamente si el servicio tiene precio variable o fijo.

📌 Servicios con precio variable (por rango de boleta)

Usá alguno de estos formatos en la descripción para que el sistema identifique el rango automáticamente al cargar un cobro:

Recomendado RANGO: 1000-5000 Etiqueta explícita, sin ambigüedad
Aceptado desde 1000 hasta 5000 Con palabras "desde" y "hasta"
Aceptado de 1000 a 5000 Con "de… a…"
Aceptado entre 1000 y 5000 Con "entre… y…"
Aceptado 1000-5000 Guión entre dos números

📦 Servicios de precio fijo

Escribí la descripción sin ninguno de esos patrones. El sistema ignorará el valor de la boleta y aplicará siempre el monto del servicio.

Fijo Internet satelital paquete B1 Sin rango → precio fijo
Fijo Netflix mensual stark-link-20 Guión en nombre → no se interpreta como rango
⚠️ Rangos solapados: si dos servicios con el mismo prefijo cubren el mismo valor (ej. EZ1: 100-300 y EZ2: 200-400 ambos cubren 250), el sistema reportará un conflicto. Revisá que los rangos no se superpongan.
